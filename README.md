## Compiling

Install the required dependencies (`sdl2` and `sdl2_ttf`, soon `sdl2_mixer` too):

| OS           | Command                                                       |
| :----------- | :------------------------------------------------------------ |
| MacOS        | `brew install sdl2 sdl2_ttf sdl2_mixer`                       |
| Linux (Arch) | `sudo pacman -S sdl2 sdl2_ttf sdl2_mixer`                     |
